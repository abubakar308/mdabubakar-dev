# 🚀 Modern Full-Stack Developer Portfolio

A high-performance, production-ready portfolio built with a cutting-edge technical stack. Designed for recruiters and technical managers to showcase engineering expertise, scalability, and premium UI/UX design.

![Portfolio Preview](./public/images/portfolio.png)

## 💎 Project Essence

This portfolio is not just a resume; it's a demonstration of modern web engineering. It features an "Antigravity" glassmorphic design language, meticulously categorized technical arsenals, and a recruiter-first project showcase.

### Key Highlights
- **High-Density Tech Stack**: Categorized skills with specific highlights for core competencies like **Prisma**.
- **Feature-Centric Projects**: Project cards emphasizing core technical challenges and solutions over generic descriptions.
- **Performance Optimized**: Leveraging Next.js 16 (Canary) and React 19 for minimal bundle size and exceptional core web vitals.
- **Premium Interactivity**: Smooth scrolling powered by **Lenis** and orchestration via **Framer Motion 12**.

## 🛠️ Technical Arsenal

### Core Frameworks
- **Next.js 16 (App Router)**: Utilizing advanced server components and lean client rendering.
- **React 19**: Benefiting from ultra-efficient state management and compiler-level optimizations.
- **TypeScript**: Strict type safety across the entire application.

### UI & Styling
- **Tailwind CSS 4**: Next-gen styling with native CSS variables and optimized build times.
- **shadcn/ui**: Accessible, high-quality component foundations.
- **Framer Motion 12**: Fluid, declarative animations for micro-interactions.
- **GSAP 3**: Complex timeline-based motion orchestration.
- **Lenis**: Industry-standard smooth scrolling.

### Backend & Data
- **Node.js & Express**: Scalable server logic foundation.
- **ORM**: **Prisma** (Primary choice for type-safe database access).
- **Databases**: PostgreSQL, MongoDB, Firebase, Supabase.

## 📁 Project Structure

```text
├── src/
│   ├── app/            # Next.js 16 App Router (Layouts, Pages, Globals)
│   ├── components/     # Reusable components (UI, Common, Navigation)
│   ├── data/           # Structured data for Projects & Skills
│   ├── sections/       # Feature sections (Hero, About, Projects, etc.)
│   └── lib/            # Utility functions and shared logic
├── public/             # Static assets (Images, Resume)
└── components.json     # shadcn/ui configuration
```

## ⚡ Getting Started

### Prerequisites
- Node.js 20+ 
- npm / pnpm / yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/abubakar308/abubakar-portfolio.git
   ```
2. Navigate to the directory:
   ```bash
   cd my-portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
